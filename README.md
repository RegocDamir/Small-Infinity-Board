# Infinity

A single-file infinite canvas editor. No build step, no dependencies — just open `index.html`.

## Usage

Open `index.html` in a browser. Everything runs locally.

## Features

- Infinite pannable/zoomable canvas
- Text, image, and video nodes
- Container and stack nodes
- Flip nodes to reveal a back face
- Spawn mode: stash nodes into a mother node, release them in an arc
- Connect nodes with lines
- State auto-saved to `localStorage`

## Deployment

Drop `index.html` (and `favicon.png`) anywhere — Netlify, GitHub Pages, or a plain file server.
